Searcher API Reference

Scholarly search, web content extraction, and PDF download service. All endpoints return JSON. This page is served live by the API at /api-reference.

Prod https://searcher.xds-lab.com/aev/search
 
Local http://127.0.0.1:8000

Interactive OpenAPI docs: <base>/docs  ·  MCP endpoint: <base>/mcp

Response Format

Search Envelope

All search endpoints wrap results in a common envelope. Exceptions: /fetch_page, /review_page, /download_pdf.

{
  "provider":               "ieeexplore",
  "query":                  "UAV trajectory optimization",
  "search_date":            "2026-05-27",
  "total_records_available": 1842,      // null for browser-scraped sources
  "total_records_downloaded": 25,
  "results": [ ... ]
}

Result Record

Every item in results uses this shape. Empty string or null when a field is unavailable. authors is always an array.

{
  "title":            "Trajectory Optimization in UAV-Assisted Cellular Networks",
  "url":              "https://ieeexplore.ieee.org/document/8714567/",
  "publication_year": 2019,
  "authors":          ["Md Moin Uddin Chowdhury", "Eyuphan Bulut", "Ismail Guvenc"],
  "doi":              "10.1109/RWS.2019.8714567",
  "source":           "2019 IEEE Radio and Wireless Symposium (RWS)",
  "snippet":          "Abstract text...",
  "pdf_link":         "https://ieeexplore.ieee.org/stamp/stamp.jsp?tp=&arnumber=8714567"
}

Environment Variables

Configure via .env (production) or .env.dev (local). Copy .env.example to get started.

ELSEVIER_API_KEY /search_scopus, /search_sciencedirect Yes Elsevier API key
SEMANTIC_SCHOLAR_API_KEY /search_semantic_scholar Optional Raises rate limits when set
BROWSER_WORKER_URL Browser-backed endpoints Yes Internal URL of the browser_worker service (default: http://127.0.0.1:8010)
VNC_PASSWORD noVNC Yes Password for the noVNC remote desktop

PDF_MAX_MB /download_pdf Optional Maximum PDF size in MB (default: 50)
REQUEST_TIMEOUT_SECONDS All HTTP endpoints Optional Per-request timeout in seconds (default: 20)

Utility Endpoints

GET
/health
Returns service status and current version string.

No parameters

Example response

{ "status": "ok", "version_name": "searcher-stack-v1.1.152" }
GET
/fetch_page
Fetch a URL and extract readable text, title, meta description, and up to 25 links.

Parameters

NameTypeDefaultDescription
url* string — Page URL to fetch
include_html bool false Include raw HTML in the response
max_chars int ≥ 500 12000 Maximum characters of body text to return

Example

GET /fetch_page?url=https://ieeexplore.ieee.org/document/8714567/

Response

{
  "url":              "https://ieeexplore.ieee.org/document/8714567/",
  "title":            "Page title from <title> tag",
  "meta_description": "Meta description content",
  "text":             "Extracted body text (up to max_chars)...",
  "links":            ["https://...", "https://...", "...up to 25 hrefs"],
  "html":             "<html>...</html>"   // only when include_html=true
}
GET
/review_page
Like /fetch_page but adds word count, estimated read time, and heading list — suited for AI review tasks.

Parameters

NameTypeDefaultDescription
url* string — Page URL to fetch
include_html bool false Include raw HTML in the response
max_chars int ≥ 500 12000 Maximum characters of body text to return

Example

GET /review_page?url=https://ieeexplore.ieee.org/document/8714567/

Response

All fields from /fetch_page, plus:

{
  "word_count":                    3200,
  "estimated_read_time_minutes":   15,
  "headings":                      ["Introduction", "Related Work", "...up to 15 h1/h2/h3"]
}
GET
/download_pdf
Stream a PDF from a direct URL and save it server-side. Does not use the browser session.
max 50 MB by default

Parameters

NameTypeDefaultDescription
url* string — Direct URL to a PDF file

Example

GET /download_pdf?url=https://arxiv.org/pdf/2104.02239

Response

{
  "path":       "/opt/searcher/downloads/2104.02239.pdf",
  "filename":   "2104.02239.pdf",
  "size_bytes": 843210
}

Error codes

  • 400 — URL does not appear to be a PDF (Content-Type mismatch)
  • 413 — PDF exceeds configured max size (PDF_MAX_MB)
  • 502 — Fetch failed (network error or bad HTTP status)

Notes

  • Validates Content-Type header; follows redirects
  • Filename derived from URL path; a short UUID suffix is appended when a file with the same name already exists
  • For subscription PDFs use the browser_worker /download_paper endpoint instead

Search Endpoints

GET
/search_semantic_scholar
Search Semantic Scholar via their public API. No API key required. Good for broad CS/STEM coverage with citation data.
No key required

Parameters

NameTypeDefaultDescription
query* string — Free-text search query
limit int ≥ 1 10 Maximum results to return

Example

GET /search_semantic_scholar?query=UAV+trajectory+optimization&limit=20

Notes

  • Does not support Boolean operators or year filtering
  • Setting SEMANTIC_SCHOLAR_API_KEY raises rate limits
GET
/search_ieeexplore
Search IEEE Xplore via the IEEE Xplore Metadata API. Supports Boolean operators and rich filtering. Auto-paginates in pages of 200.
Requires IEEE_XPLORE_API_KEY Paginated · 1-based

Parameters

NameTypeDefaultDescription
query* string — Search query. Supports AND, OR, NOT Boolean operators
limit int ≥ 1 25 Maximum results to return. The service auto-paginates in batches of 200.
start_record int ≥ 1 1 1-based offset into the result set. Use for manual pagination.
year_low int null Earliest publication year (inclusive)
year_high int null Latest publication year (inclusive)
content_type string null Filter by type: Books, Conferences, Courses, Early Access, Journals, Magazines, Standards
open_access bool null true restricts to open-access articles only. false / omitted = no filter.
sort_field string null One of: article_number, article_title, publication_title. Invalid values return HTTP 422.
sort_order string null asc or desc. Invalid values return HTTP 422.
author string (min 3 chars) null Filter by author first or last name. Fewer than 3 characters returns HTTP 422.

Examples

# Filtered journal search
GET /search_ieeexplore?query=UAV+AND+trajectory&limit=50&year_low=2018&content_type=Journals

# Manual pagination — records 201–400
GET /search_ieeexplore?query=UAV+AND+trajectory&limit=200&start_record=201

Notes

  • Returns HTTP 400 when IEEE_XPLORE_API_KEY is not configured
  • pdf_link points to the IEEE stamp URL; full-text access depends on institutional subscription
  • open_access=false and omitting open_access both produce unfiltered results — only true applies a filter
GET
/search_scopus
Search Scopus (Elsevier) via the Scopus Search API. Supports full Scopus field-code Boolean syntax. Auto-paginates in pages of 25.
Requires ELSEVIER_API_KEY Paginated · 0-based

Parameters

NameTypeDefaultDescription
query* string — Scopus query. Field codes: TITLE(), ABS(), AUTH(), TITLE-ABS-KEY(), etc.
limit int ≥ 1 500 Maximum results to return
start int ≥ 0 0 0-based result offset
year_low int null Earliest publication year (inclusive)
year_high int null Latest publication year (inclusive)
subj string ENGI Scopus subject area code. Common: ENGI, COMP, MATE, PHYS. Default is ENGI and is always applied unless overridden.

Examples

# Field-code query with year filter
GET /search_scopus?query=TITLE-ABS-KEY(UAV+trajectory)&limit=100&year_low=2018&subj=ENGI

# Second page of 25
GET /search_scopus?query=TITLE-ABS-KEY(UAV)&limit=25&start=25
GET
/search_sciencedirect
Search ScienceDirect (Elsevier) full-text journals via the Elsevier API.
Requires ELSEVIER_API_KEY Paginated · 0-based

Parameters

NameTypeDefaultDescription
query* string — Free-text search query
limit int ≥ 1 20 Maximum results to return
start int ≥ 0 0 0-based result offset
year_low int null Earliest publication year (inclusive)
year_high int null Latest publication year (inclusive)

Example

GET /search_sciencedirect?query=UAV+trajectory+optimization&limit=30&year_low=2019

Notes

  • Covers Elsevier full-text articles; complements Scopus which covers abstracts across many publishers
GET
/search_google_scholar_browser
Search Google Scholar by driving the persistent Chromium browser. Handles CAPTCHAs via the shared noVNC session.
Requires browser_worker Paginated · 0-based

Parameters

NameTypeDefaultDescription
query* string — Search query
limit int ≥ 1 200 Maximum results to return
start_index int ≥ 0 0 0-based result offset for pagination
year_low int null Earliest publication year (inclusive)
year_high int null Latest publication year (inclusive)
exclude_domains list[string] null Domains to exclude. Default excludes researchgate.net, books.google.com, search.proquest.com. Pass [] to disable filtering. Repeat the parameter for multiple values.

Example

GET /search_google_scholar_browser?query=UAV+path+planning&limit=50&year_low=2020

Notes

  • total_records_available is always null — Google Scholar does not expose a total count
  • If a CAPTCHA appears, open noVNC, solve it once — the browser session persists
  • pdf_link is populated only when Google Scholar surfaces a direct PDF link
GET
/search_web_of_science
Search Web of Science by driving the persistent Chromium browser. Requires an active institutional WoS session.
Requires browser_worker + WoS session

Parameters

NameTypeDefaultDescription
query* string — Search query
limit int ≥ 1 10 Maximum results to return
year_low int null Earliest publication year (inclusive)
year_high int null Latest publication year (inclusive)

Example

GET /search_web_of_science?query=UAV+trajectory+optimization&limit=20&year_low=2019

Notes

  • Log in to Web of Science via noVNC before use if the session has expired
  • total_records_available may be null depending on WoS page structure
GET
/search_ebsco
Search EBSCO Research by driving the persistent Chromium browser. Requires an active institutional EBSCO session.
Requires browser_worker + EBSCO session

Parameters

NameTypeDefaultDescription
query* string — Search query
limit int ≥ 1 100 Maximum results to return
year_low int null Earliest publication year (inclusive)
year_high int null Latest publication year (inclusive)

Example

EBSCO Download Endpoints

GET
/download_ebsco_paper
Download a single paper from an EBSCO detail page URL using the browser.
Requires browser_worker + EBSCO session

Parameters

NameTypeDefaultDescription
url* string — EBSCO paper detail page URL (e.g. https://research.ebsco.com/c/.../search/details/...)

Example

GET /download_ebsco_paper?url=https://research.ebsco.com/c/abc123/search/details/xyz789

Notes

  • Navigates to the detail page, clicks Download twice (first opens popup, second confirms), then captures and saves the PDF
  • Response is proxied from browser_worker — typically { "path": "...", "filename": "...", "size_bytes": ... }
POST
/download_ebsco_papers
Download multiple EBSCO papers sequentially from a list of detail page URLs.
Requires browser_worker + EBSCO session

Request body

JSON array of EBSCO detail page URL strings.

[
  "https://research.ebsco.com/c/.../search/details/url1",
  "https://research.ebsco.com/c/.../search/details/url2"
]

Notes

  • Each URL is processed in order; a failure on one URL does not stop the batch
  • Request timeout scales with URL count: 60s × n + 30s
  • Response is proxied from browser_worker — a dict with completed downloads and any failures

Common Patterns for AI Agents

Which search endpoint should I use?

GoalRecommended endpoint(s)
Peer-reviewed engineering / CS literature/search_ieeexplore, /search_scopus
Elsevier full-text articles/search_sciencedirect
Broad coverage incl. preprints & grey lit/search_google_scholar_browser
Quick query, no API key/search_semantic_scholar
WoS citation tracking/search_web_of_science
EBSCO database access/search_ebsco

Retrieve paper details from a result URL

GET /fetch_page?url={result.url}
# Or for AI review tasks:
GET /review_page?url={result.url}

Download a paper

# Open-access direct PDF link:
GET /download_pdf?url={result.pdf_link}

# Subscription paper (IEEE, Elsevier, etc.) — use browser_worker:
POST https://searcher.xds-lab.com/aev/browser/download_paper
Content-Type: application/json
{ "url": "{result.url}" }

Paginate through results

# IEEE — 1-based start_record
GET /search_ieeexplore?query=UAV&limit=200&start_record=1
GET /search_ieeexplore?query=UAV&limit=200&start_record=201

# Scopus / ScienceDirect — 0-based start
GET /search_scopus?query=UAV&limit=25&start=0
GET /search_scopus?query=UAV&limit=25&start=25

# Google Scholar browser — 0-based start_index
GET /search_google_scholar_browser?query=UAV&limit=10&start_index=0
GET /search_google_scholar_browser?query=UAV&limit=10&start_index=10
